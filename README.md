# 基于Python的期货程序化交易系统 python1138

## 项目概述

本项目是一个基于Python语言的期货程序化交易系统，通过前后端分离的方式，提供用户友好的交互界面和高效的后端管理能力。

## 技术栈

- Python
- Flask（Web开发框架）
- MySQL（数据库）

## 功能模块

### 前台功能

- 首页：展示系统概览
- 期货信息：提供期货相关信息
- 投资风险：揭示投资风险
- 资讯信息：发布相关资讯
- 意见反馈：收集用户反馈

### 管理员功能

- 首页：后台管理概览
- 期货公司管理：管理期货公司信息
- 开户信息管理：处理用户开户信息
- 充值信息管理：管理用户充值记录
- 期货类型管理：维护期货种类信息
- 期货信息管理：更新与维护期货信息
- 我的信息：管理员个人信息管理
- 用户管理：管理系统用户

## 系统角色

- 前台用户
- 管理员
